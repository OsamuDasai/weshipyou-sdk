# WeShipYou SDK Documentation

Bienvenido a la documentación oficial del SDK empresarial para **WeShipYou**. Este paquete te permite integrar envíos internacionales, recargas móviles y gestión de cuentas directamente desde tu aplicación Node.js/TypeScript con cero configuración manual.

## ✨ Características Clave

*   🚀 **API Client Pre-configurado**: Autenticación automática, reintentos exponenciales y manejo de errores nativo.
*   🛡️ **Resiliencia**: Circuit Breakers integrados (`Opossum`) y polling inteligente para tracking.
*   🔐 **Multi-Tenant**: Aislamiento de datos por cliente usando `AsyncLocalStorage`.
*   📊 **Observabilidad**: Tracing distribuido con OpenTelemetry y métricas listas para Prometheus.
*   ⚙️ **CLI & Scaffolding**: Genera proyectos completos con un solo comando.

## 🚀 Comenzar rápido

```bash
npm install weshipyou-sdk
```

Para más detalles sobre cómo configurar el entorno y tus primeras llamadas, ve a [Instalación](installation.md).
